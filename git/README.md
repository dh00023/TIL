# git

git과 관련된 글들을 참조하여 정리한 repository입니다.

## 버전관리 시스템이란?

- 파일이름을 더럽히지 않는 버전관리.
- **복원**, **백업**, **협업**
- 작업의 지배자
- CVS, SVN, **GIT**

버전관리를 한다는 것은 프로젝트 폴더(저장소)에 생성되는 파일들을 체계적으로 관리하겠다는 것이다.
